Configuration files for developer's Apache Httpd service.
---------------------------------------------------------

See the Getting Started Guide (on Google drive) for installation details.

See conf/conf.d/httpd-vhosts.conf for the virtual host definition and httpd-ssl.conf
for the https version.

See httpd-shibboleth.conf for details of starting up this dev version of Apache HTTPD
with Shibboleth security.  By default it will start with a mock security installation.
DO NOT DEPLOY THIS TO LIVE/TEST/anywhere except developer machines.
